# CIPRO Dataset and Resources

This repository hosts the **dataset and supporting resources** used in our paper:

> **CIPRO: A Pixel-Row Oversampling-Based Approach for Code Image Enhancement in Vulnerability Detection**  
> (Currently under review)

⚠️ **Note:** At this stage, only the **datasets and resources** are publicly available.  
The implementation code will be released after the paper is formally accepted.

---
